Question 1: (Solve using Elimination Method)

The elimination method works by:
  • Making the coefficients of one variable the same in both equations
  • Adding or subtracting the equations to eliminate that variable
  • Solving for the remaining variable
  • Substituting back to find the other variable

Question 2: (Solve using Substitution Method)
2y = 2x - 3
3y = x - 1
Solution:
üìå Substitution Method Overview:
The substitution method works by:
  • Rearranging one equation to make one variable the subject
  • Substituting this expression into the other equation
  • Solving the resulting equation with one variable
  • Substituting back to find the other variable
Step 1: Label the equations
Equation ‚ë†: 2y = 2x - 3
Equation ‚ë°: 3y = x - 1
Label the equations for reference
Step 2: Rearrange equation ‚ë° to make x the subject
3y = x - 1
x = 3y + 1
Add 1 to both sides to isolate x
Step 3: Substitute x = 3y + 1 into equation ‚ë†
2y = 2x - 3
2y = 2(3y + 1) - 3
Replace x with the expression we found
Step 4: Expand the brackets
2y = 2(3y + 1) - 3
2y = 6y + 2 - 3
2y = 6y - 1
Multiply out and simplify
Step 5: Solve for y
2y = 6y - 1
2y - 6y = -1
-4y = -1
y = -1 √∑ -4
y = 1/4
Collect y terms on one side and divide
Step 6: Substitute y = 1/4 back into x = 3y + 1
x = 3y + 1
x = 3(1/4) + 1
x = 3/4 + 1
x = 3/4 + 4/4
x = 7/4
Use the expression for x to find its value
Step 7: Check the solution in equation ‚ë†

2(1/4) = 2(7/4) - 3

Verify our solution works in the original equation
