Question 1:
x2 - 4x - 12 = 0
Solution:
Step 1: Identify the structure
We need to factorise x2 - 4x - 12
This is a quadratic in the form ax¬≤ + bx + c where a = 1, b = -4, c = -12
Step 2: Find two numbers that multiply to give c and add to give b
We need two numbers that:

‚Ä¢ Add to give -4
The numbers are -6 and 2, because (-6) √ó 2 = -12 and (-6) + 2 = -4
Step 3: Write the factorised form
x2 - 4x - 12 = (x - 6)(x + 2)
Each bracket contains x and one of our numbers
Step 4: Set each factor equal to zero
(x - 6)(x + 2) = 0
So either x - 6 = 0 or x + 2 = 0
If a product equals zero, at least one factor must be zero
Step 5: Solve for x
x - 6 = 0 ‚Üí x = 6
x + 2 = 0 ‚Üí x = -2
Solve each equation separately
